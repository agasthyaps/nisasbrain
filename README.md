## nisa: Expertise Repo (private)

nisa, this private expertise repo is your differentiator—use it constantly. Users come to you because you synthesize their context with these resources. Consult it especially for abilities 2, 3, 4, 5, and 7.

### Navigate
- `getExpertiseTree`: repository tree
- `listExpertiseFiles`: list files/dirs in a path
- `readExpertiseFile`: read file contents
- `searchExpertiseContent`: search by name/content
- `getExpertiseOverview`: return this README

### Contents

#### **Foundational**
Core teaching and coaching practices that apply across all contexts and subjects.
- `foundational/10_key_teaching_moves.md` — foundational teaching moves to be adapted across subject areas
- `foundational/warm_demander.md` — principles of being a warm demander
- `foundational/coaching_moves.md` — key coaching moves for effective instructional coaching
- `foundational/foundational_skills_observation_tool.md` — K-2 observation tool for phonics, phonological awareness, print concepts, and fluency instruction. "core action" usually refers to something in here.
- `foundational/tlac_strategies.md` — Teach Like a Champion 2.0's 62 techniques for effective classroom instruction

#### **Math Teaching Practices**
Math-specific instructional practices, questioning strategies, and observation tools.
- `math_teaching_practices/key_math_teaching_practices.md` — comprehensive math teaching moves for elementary and middle/high school
- `math_teaching_practices/rigorous_math_questions.md` — question stems to promote mathematical discourse
- `math_teaching_practices/ipg_math_k-8.md` — Instructional Practice Guide and core actions (IPG) for observing K-8 math lessons (focus, coherence, rigor)
- `math_teaching_practices/ipg_math_hs.md` — Instructional Practice Guide and core actions (IPG) for observing high school math lessons (focus, coherence, rigor)

#### **ELA Teaching Practices**
ELA-specific instructional practices and observation tools.
- `ela_teaching_practices/key_ela_teaching_practices.md` — ELA teaching moves for middle/high school (elementary content to be added)
- `ela_teaching_practices/ipg_ela_k-2.md` — Instructional Practice Guide and core actions (IPG) for observing K-2 ELA/literacy lessons (reading/listening comprehension)
- `ela_teaching_practices/ipg_ela_3-12.md` — Instructional Practice Guide and core actions (IPG) for observing grades 3-12 ELA/literacy lessons

#### **Data Protocols**
Frameworks and protocols for data-driven instruction.
- `data_protocols/relay_data_protocol.md` — Relay Graduate School data protocol
- `data_protocols/datawise.md` — 8-step inquiry process for data analysis
- `data_protocols/data_driven_dialogue.md` — dialogue protocols for data discussions

#### **Other Vetted Tools**
External tools and resources that complement nisa's capabilities.
- `other_vetted_tools/coteach.md` — AI planning assistant for IM curriculum (coteach.ai)
- `other_vetted_tools/writingpathway.md` - research-backed AI-powered tool for targeted writing practice and assessment.

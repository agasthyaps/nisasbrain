# CoTeach

## Overview
CoTeach (coteach.ai) is an AI-powered planning assistant built by Teaching Lab Studio that helps educators prepare effective, grade-level instruction. It's designed to support teachers to implement curriculum-aligned scaffolds and supports that ensure all students can access rigorous learning.

## Key Features
- **Direct IM v.360 Integration**: All responses are aligned with the instructional approach of Illustrative Math v.360, and you can adjust curriculum content directly within CoTeach
- **Visual Aids**: Able to create accurate diagrams and visual aids
- **Scaffolding Support**: Helps teachers differentiate, adapt, and edit IM lessons to meet diverse student needs
